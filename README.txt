# Character Strength Calculator

## Description
This program is a Character Strength Calculator based on the CH Base Magic System's Power Level Rank. It allows you to input character stats and calculates various levels and rankings based on those stats.

## Features
- Calculate Physical Level
- Calculate Magical Power
- Calculate Mental Power
- Calculate Power Level
- Calculate Number Rank Level

## Usage
To use the Character Strength Calculator, follow the options provided in the user interface. Here are the available options:

0. Quit: Exit the program.
1. Input Character Stats: Enter the character's offensive strength, defensive strength, speed, agility, mana capacity, maximum mana output, mana regeneration rate, IQ, battle IQ, and knowledge scores to calculate the character's levels and rankings.
2. View Character Stats & Levels: (Feature not implemented yet)
3. Edit Character Stats: (Feature not implemented yet)
4. Save Character Data: (Feature not implemented yet)
5. Load data from file: (Feature not implemented yet)

## Installation and Setup
- Ensure you have Python installed on your system.
- Clone the repository or download the code files.
- Install the required dependencies.
- Run the program using a Python interpreter.
