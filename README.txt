# Character Strength Calculator

## Description
This program is a Character Strength Calculator based on the CH Base Magic System's Power Level Rank. It allows you to input character stats and calculates various levels and rankings based on those stats.

## Features
- Calculate Physical Level
- Calculate Magical Power
- Calculate Mental Power
- Calculate Power Level
- Calculate Number Rank Level

## Usage
To use the program, follow these steps:

Make sure you have Python installed on your system.
Run the main.py file using the Python interpreter.
The program will display a menu with several options:
0: Quit: Exit the program.
1: Input Character Stats: Enter the stats of a character to calculate their strength.
2: View all saved Characters: View the details of all saved characters.
3: View a Character: Enter the name of a character to view their details.
4: Edit Character Stats: Enter the name of a character to edit their stats.
5: Save or Load from .txt file: Save or load characters from a text file.
Select the desired option by entering the corresponding number.
Follow the prompts to input data, view characters, edit stats, or save/load from a file.

##Character Class
The program defines a Character class that represents a character. Each character has a name and various attributes such as offensive strength, defensive strength, speed, agility, mana capacity, and more. The class provides methods to add spells to a character, view their spells, and print their details.

##Calculation Functions
The program includes several functions to calculate different levels of a character's strength based on their attributes. These functions are used internally by the Character class.

##File Operations
The program includes placeholder functions for reading and writing character data to a file. However, these functions are not implemented yet.
